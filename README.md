# Local From Bocas

Sitio web oficial de **Local From Bocas**, una marca turística que conecta a los viajeros con la auténtica cultura y belleza de Bocas del Toro, Panamá.

## 🏝️ Sobre el Proyecto

Local From Bocas ofrece experiencias turísticas de alta calidad, productos culturales auténticos y asistencia personalizada para viajeros en Bocas del Toro. Nuestro sitio web genera contenido de valor para aumentar el engagement, atraer seguidores e impulsar las ventas.

## 🚀 Características

- **Tours Históricos**: Experiencias educativas y entretenidas
- **Mercancía Cultural**: Productos de alta calidad con diseños originales
- **Asistencia Personalizada**: Servicio de alta calidad para viajeros
- **Contenido de Valor**: Artículos sobre Bocas del Toro y Panamá
- **Información Completa**: Datos de las 10 provincias y 3 comarcas de Panamá

## 🛠️ Tecnologías

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Hosting**: Railway
- **Control de Versiones**: GitHub
- **Diseño**: Sistema de diseño personalizado

## 🚀 Desarrollo Local

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/[username]/local-from-bocas.git
   cd local-from-bocas
   ```

2. **Iniciar servidor local**:
   ```bash
   npm start
   # o
   python3 -m http.server 8000
   ```

3. **Abrir en el navegador**:
   ```
   http://localhost:8000
   ```

## 🌐 Deployment

### Railway
El sitio se despliega automáticamente en Railway desde la rama `main`:

1. **Configuración automática**: Railway detecta el proyecto estático
2. **SSL**: Certificado automático
3. **CDN**: Distribución global incluida
4. **Custom Domain**: Configuración de DNS

### GitHub Pages (Alternativa)
```bash
# Configurar GitHub Pages
git subtree push --prefix local-from-bocas origin gh-pages
```
